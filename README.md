# HR Assistant Agent

## Overview
AI bot for HR queries (policies, leave, benefits). Uses FAQs + GPT.

## Features
- Keyword FAQ matching.
- AI-enhanced responses.
- Chat UI.

## Limitations
- Static FAQs (no DB).
- Basic search.

## Tech
- OpenAI GPT-3.5, Gradio.

## Run
pip install openai gradio
python app.py
